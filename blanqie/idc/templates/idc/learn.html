How it works...

  • Prepare a text file with content you want to be tested on
  • Go to the Test tab
  • Click on upload the file area to choose a file
  • Select the file you prepared
  • Click on the 'See BLANQIE' button
  • The context from the text file will appear on the page with randomized words blanked out
  • You can start testing your knowledge!

**NOTE: You must attach a text file or the program will not work. If you press the 'See BLANQIE' button without attaching a file then you will get an error
